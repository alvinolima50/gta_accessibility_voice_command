Acessibilidade GTA

Configuração de comandos de voz

Como usar

  • Palavras-chave: uma por linha. O sistema identifica se qualquer uma delas aparece na sua fala.
  • Teclas especiais: shift, space, enter, tab, ctrl, alt, esc, f1–f12, setas.
  • Mouse: mouse_left, mouse_right, mouse_middle.
  • Letras/números: use só o caractere. Ex: f, b, 1.
  • Para testar a tecla sem falar, clique em Testar. Dê foco no GTA antes (alt-tab).
